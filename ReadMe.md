## Haiku-CMake

A CMake module to add extra functionality for the Haiku operating system.  It provides functions for adding application resources, BFS attributes, localization catalogs, and more...

### Included Project Examples
  * [HelloHaiku](examples/HelloHaiku) - a basic windowed application with extra BFS attributes
  * [HelloHaikuI18n](examples/HelloHaikuI18n) - a basic windowed application with translation support
